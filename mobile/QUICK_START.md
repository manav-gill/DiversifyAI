# 🚀 PortfolioPilot Mobile - Quick Start Guide

## ⚡ 30-Second Quick Start

```bash
cd PortfolioPilotMobile
npm start
```

Then press:
- `a` - Run on Android Emulator
- `i` - Run on iOS Simulator  
- `w` - Run on Web Browser
- `r` - Reload
- `d` - Open DevTools

## 📱 What You'll See

A centered screen displaying:
```
PortfolioPilot Mobile Running ✅

[Test API]
```

## 🧪 Test the API

1. Make sure backend is running: `npm start` in backend folder
2. Click the "Test API" button on the app
3. See connection status displayed

## 🔧 Configuration

**Backend API Base URL**: `http://localhost:5000/api`  
Location: `src/services/api.js` line 3

For Android emulator use: `http://10.0.2.2:5000/api`

## 🚨 Troubleshooting

| Issue | Solution |
|-------|----------|
| Port in use | Kill process or use `npm start -- --port=8082` |
| Module not found | Run `npm install` again |
| Cache issues | Run `npm start -- --reset-cache` |
| Android not found | Ensure Android Studio is installed |
| API not connecting | Check backend is running on port 5000 |
